# 性别判断器

这是一个用 Python 和 Tkinter 写的简单 GUI 程序，功能是根据用户输入的名字随机显示“你是男的”或“你是女的”。

---

## 功能介绍

- 启动后弹出窗口，提示输入名字
- 点击“确认”按钮后，随机显示性别（男或女）
- 简单易用，适合学习 GUI 编程和打包发布

---

## 运行环境

- Python 3.6 及以上版本（开发环境）
- Windows 10/11（运行环境）
- 依赖库：Tkinter（Python 标准库，无需额外安装）

---

## 使用说明

### 源码运行

1. 确保安装 Python 3.6+  
2. 在命令行执行：  
   python main.py
